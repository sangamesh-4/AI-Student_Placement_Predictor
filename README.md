## 🚀 Overview

AI Placement Predictor is a Machine Learning-powered web application that predicts whether a student is likely to get placed based on academic performance and skillset.

The application also provides **personalized suggestions** to help improve placement chances.

---

## ✨ Key Features

- 🎯 Real-time Placement Prediction
- 🎨 Interactive & User-Friendly Dashboard
- 💡 Personalized Improvement Suggestions
- 📈 Profile Strength Visualization (Radar Chart)
- 🧠 Smart Feedback System
- 🌐 Deployed on Streamlit Cloud

---

## 🧠 How It Works

1. User inputs student details  
2. Data is preprocessed (encoding + scaling)  
3. ML model predicts placement outcome  
4. App generates suggestions based on profile  

---

## 📊 Input Parameters

- IQ  
- CGPA  
- Previous Semester Result  
- Academic Performance  
- Internship Experience  
- Extra Curricular Activities  
- Communication Skills  
- Projects Completed

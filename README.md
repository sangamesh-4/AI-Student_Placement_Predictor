## 🚀 Overview

AI Placement Predictor is a Machine Learning-based web application that predicts whether a student is likely to get placed based on academic performance and skillset.

The app also provides **personalized suggestions** to help users improve their placement chances.

---

## ✨ Features

- 🎯 Placement Prediction (Placed / Not Placed)
- 📊 Interactive Streamlit Dashboard
- 💡 Personalized Career Suggestions
- 📈 Profile Strength Visualization
- 🎨 Clean and User-Friendly UI

---

## 🧠 How It Works

1. User enters student details
2. Input data is preprocessed (encoding + scaling)
3. Trained ML model predicts placement outcome
4. App provides actionable suggestions

---

## 📊 Input Parameters

- IQ
- CGPA
- Previous Semester Result
- Academic Performance
- Internship Experience
- Extra Curricular Activities
- Communication Skills
- Projects Completed
